# 李昕芸 — 交互设计作品集

一个纯静态的个人设计作品集网站，面向 B2B 交互设计师，展示项目案例与联系方式。

## 技术栈

- **HTML** — 页面结构
- **CSS** — 样式（CSS 变量实现主题切换）
- **JavaScript** — 交互逻辑（原生 JS，无框架依赖）

没有用任何框架、构建工具或后端服务，直接双击 index.html 就能在浏览器中打开。

## 项目结构

```
├── index.html          # 页面入口
├── css/
│   └── style.css       # 所有样式（主题变量、布局、动画）
├── js/
│   └── main.js         # 所有交互（主题切换、滚动动画、表单）
├── assets/
│   └── images/         # 图片资源（项目截图、logo）
└── icons/              # SVG 图标库（备用）
```

职责分离清晰：改样式找 css/，改逻辑找 js/，换图片进 assets/images/，互不影响。

## 页面设计思路

**整体基调：克制、专业、留白**

作为一名 B2B 交互设计师的作品集，设计本身也是作品的一部分。整体走极简路线，用大量留白和克制的配色表达「少即是多」的设计理念，符合企业级产品设计的审美预期。

**配色方案**

- 暗色模式（默认）：深黑底色 #0d0d0d，蓝调强调 #5b8def
- 亮色模式：暖白底色 #fafaf8，蓝调强调 #3b6fd4
- 两种主题使用完全独立的 CSS 变量，一键切换

**版式节奏**

页面采用纵向单页叙事，从上到下依次为：Hero 大标题 → 项目案例 → 个人简介 → 联系方式。每个区块撑满一屏，浏览时自然停顿，节奏感强。内容区最大宽度 1200px，两端留白透气。

**微交互**

- 主题切换：sun/moon 图标带旋转动画切换
- 滚动进场：页面元素随滚动渐显上移，引导视线流动
- 导航栏：向下滚动自动隐藏，回滚复现，不遮挡内容
- 项目卡片：hover 时图片微放大，浮现预览/跳转按钮
- 按钮：hover 有微位移反馈，强化可点击感知
- 邮箱复制：点击一键复制，图标切换为绿色对勾

**字体**

使用系统原生字体栈（SF Pro / PingFang SC / Microsoft YaHei），无需加载外部字体文件，加载速度极快，且在各平台呈现最佳原生阅读体验。

## 如何部署

直接拖整个文件夹到 Vercel / Netlify / GitHub Pages 即可，零配置。
